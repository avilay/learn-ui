Flex Paragraph Examples

Paragraphs

Flex items without any basis set

All the flex items are 0 0 auto, which means that they won't grow or shrink and will take on their "natural" width.

  • The first item is fixed-width and has a fixed-width child element. The child is wider than the parent, which is why content is overflowing.
  • The second item does not have any width specified, but has a fixed-width child. The item takes on the width of the child.
  • For the third item, neither the item itself, nor its child have any width specified. The child takes on the max-width of the #text, and the parent flex item takes on the width of its child.

Fixed width item with fixed width para: It was the best of times, it was the worst of times, it was the age of wisdom, it was the age of foolishness...

No width item with fixed width para: it was the epoch of incredulity

No width item with with no width para: it was the epoch of belief

Flex items with basis

All the flex items are 0 0 400px, which means again that they won't grow or shrink but will try to keep to 400px.

  • The first item has a wider fixed width child, so it expands to take on the width of its child.
  • The second item has a narrower fixed width child, it keeps its basis width and the child keeps its fixed width.
  • The third item has a no width child with the leaf child being #text, so they all keep the basis width with the text becoming responsive.

Basis item with wider fixed width para: It was the best of times, it was the worst of times, it was the age of wisdom, it was the age of foolishness...

Basis item with narrower fixed width para: it was the epoch of incredulity

Basis item with with no width para: it was the epoch of belief
