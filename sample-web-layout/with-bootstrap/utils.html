Bootstrap Utilities

This page demonstrates some of the harder to grok utilities in bootstrap. There are many more that are easy to understand from the documentation that are not demonstrated here. Check out the full list in Bootstrap's Utilities documentation.

Floating Divs

This demos the .float-end class. It causes the image to "float" to the right. I have used .float-md-end instead of .float-end so that I can see how it would look if float-end was not given by narrowing the screen size to less than -md. For some reason whenever I am messing around with the floats I need to set the .clearfix style in the parent. There is some discussion on Stackoverflow on this.

As you can see the paragraphs gracefully wrap around the floated image. Now imagine how this would look with some actual content in here, rather than just this boring placeholder text that goes on and on, but actually conveys no tangible information at. It simply takes up space and should not really be read.

And yet, here you are, still persevering in reading this placeholder text, hoping for some more insights, or some hidden easter egg of content. A joke, perhaps. Unfortunately, there's none of that here.
